## 关键技术点

1. **Vercel AI SDK data stream format**: 前端需要解析 AI SDK 的流式协议（0: text, 9: tool call, a: tool result 等）
2. **JSON Schema → Zod**: MCP tools 的 inputSchema 是 JSON Schema 格式，需转为 Zod schema 供 AI SDK 使用
3. **SSE in Nitro**: Nitro 的 event handler 需要使用 `setResponseHeader` + `sendStream` 模式
4. **Skills 文件读取**: server 端读取 skills/ 目录下的文件，需要处理路径
5. **maxSteps 限制**: 防止 agent 无限循环，设置上限 20 步
