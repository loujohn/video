# 视觉风格与 AI 图片/视频生成指南

## 一、题材色彩体系

### 甜宠

**主色**：暖色系——粉、橙、金、米白。

**氛围**：明亮、柔和、梦幻。

**中文风格词**：`暖色调，柔粉与蜜桃色系，黄金时段般温暖，梦幻色彩`

**英文风格词**：`warm pastel tones, soft pink and peach palette, golden hour warmth, dreamy color grading`

**光影词**：`柔和逆光，暖色光晕，金色散射阳光，温柔轮廓光` / `soft backlight, warm lens flare, diffused golden sunlight, gentle rim lighting`

---

### 霸道总裁

**主色**：黑、灰、金、深蓝。

**氛围**：冷峻、高级、权力感。

**中文风格词**：`去饱和冷色调配金色点缀，暗调奢华感，深蓝与炭灰色系`

**英文风格词**：`desaturated cool tones with gold accents, dark luxury palette, muted navy and charcoal`

**光影词**：`硬光塑造轮廓，戏剧性明暗对比，尖锐阴影边缘，自上方打光的权力感` / `hard key light, dramatic chiaroscuro, sharp shadow edges, power lighting from above`

---

### 悬疑

**主色**：冷灰、深蓝、墨绿、暗红。

**氛围**：阴郁、压抑、不安。

**中文风格词**：`冷色调去饱和处理，深蓝绿色阴影，暗红点缀，黑色电影色系`

**英文风格词**：`cold desaturated grading, deep blue-green shadows, muted crimson accents, noir palette`

**光影词**：`低照度，高对比，阴影占画面60%，单一硬光源，体积雾` / `low-key lighting, heavy shadows covering 60% of frame, single harsh light source, volumetric fog`

---

### 战神 / 赘婿 / 逆袭

**主色**：偏冷、低饱和，黑灰为主，金色/红色作点缀。

**氛围**：硬朗、克制、爆发时对比强烈。

**中文风格词**：`低饱和冷色调，灰调为主，高潮时刻突现金色/红色`

**英文风格词**：`subdued cold tones, desaturated greys, punctuated by vivid gold/red in climax moments`

**光影词**：`日常偏冷平光，打脸/逆袭瞬间突然提亮加暖色` / `flat cold lighting for daily scenes, dramatic warm burst during reveal moments, contrast shift`

---

### 宫斗 / 穿越

**主色**：朱红、金、青、墨。

**氛围**：华贵、压抑、权谋感。

**中文风格词**：`帝王红金配色，翡翠绿点缀，墨黑阴影，中国传统色系`

**英文风格词**：`imperial red and gold, jade green accents, ink black shadows, traditional Chinese color palette`

**光影词**：`烛光暖色，戏剧性舞台光线，华丽织物质感捕光，灯笼柔光` / `candlelight warmth, dramatic theatrical lighting, rich fabric textures catching light, paper lantern glow`

## 二、光线设计原则

| 原则 | 说明 | 中文提示词 | 英文提示词 |
|------|------|-----------|-----------|
| 主光明确 | 每场戏有清晰主光源方向 | 主光从左上方45度角打入 | `key light from upper-left at 45 degrees` |
| 轮廓光 | 重要角色用轮廓光分离背景 | 强轮廓光将人物从背景分离 | `strong rim light separating subject from background` |
| 情绪光 | 暖光=安全，冷光=危机 | 暖色琥珀补光 / 冷蓝色顶光 | `warm amber fill light` / `cold blue overhead light` |
| 反差控制 | 冲突场景加大明暗对比 | 高对比度光比4:1 | `high contrast ratio 4:1` |

### 光线类型速查

| 光线类型 | 情绪 | 中文提示词 | 英文提示词 |
|---------|------|-----------|-----------|
| 柔光 | 温暖、安全、浪漫 | 柔和散射光，无硬阴影 | `soft diffused lighting, no harsh shadows` |
| 硬光 | 紧张、权力、对抗 | 硬光方向性照明，尖锐阴影 | `hard directional light, sharp shadow edges` |
| 逆光 | 神秘、浪漫、孤独 | 强逆光形成剪影/轮廓光 | `strong backlight creating silhouette/rim glow` |
| 侧光 | 层次、内心挣扎 | 戏剧性侧光，半明半暗 | `dramatic side lighting, half face in shadow` |
| 顶光 | 压迫、审判、不安 | 头顶硬光，眼窝深陷阴影 | `overhead harsh light, deep eye shadows` |
| 底光 | 恐怖、诡异 | 从下方打光，恐怖风格向上阴影 | `underlit face, horror-style upward shadows` |

## 三、构图规则

| 规则 | 说明 | 中文提示词 | 英文提示词 |
|------|------|-----------|-----------|
| 三分法 | 重要元素放在三分线或交点 | 三分法构图，主体在右侧三分线 | `rule of thirds, subject at right third` |
| 留白 | 情绪高潮时大量留白突出主体 | 极简构图，大面积留白 | `minimalist composition, vast negative space` |
| 景深 | 虚化背景突出主体 | 浅景深，F1.4，奶油般虚化背景 | `shallow depth of field, f/1.4, creamy bokeh` |
| 对称 | 权力场景用对称 | 对称构图 | `symmetrical composition` |
| 引导线 | 利用线条引导视线 | 引导线汇聚到主体 | `leading lines converging on subject` |
| 框中框 | 利用门窗形成嵌套构图 | 透过门框/窗框构图 | `framed through doorway, frame within frame` |

## 六、角色一致性维护方法

角色一致性是 AI 短剧生成的核心挑战。

### 参考图锚定法（首选）

1. **生成角色参考图**：为每个角色生成 6 角度参考图（见 `character-image-guide.md`）
2. **确认并锁定**：审核参考图质量，确认后标记为 `confirmed`
3. **所有后续引用参考图**：在提示词中使用 @标签 引用

### 提示词锚定法（无参考图时的备选）

在每个提示词中重复角色的 **5 个以上核心外貌特征**：

```
[角色名]：[年龄段] [性别]，[发型+发色]，[标志性面部特征]，[服装]，[气质]
```

**关键原则**：
- 每次提及角色都使用完全相同的特征描述词
- 使用具体词汇（`柳叶眉、杏眼、右颊酒窝` > `漂亮的女人`）
- 服装保持固定，换装时为每套服装建独立的 character_look
- 角色间保持高区分度——发色、服装、配饰必须明显不同

### Seedance 2.0 角色一致性专项约束

每个涉及角色的提示词必须包含：

```
同一角色面部稳定不变形，服装一致，发型不变，
人体结构正常，比例自然，动作不僵硬。
```

## 七、项目风格圣经（Style Bible）

每个项目在开始视觉生成前，建立一份**风格圣经**，作为所有提示词的**固定后缀**。

### 风格圣经模板

```json
{
  "project_style_bible": {
    "visual_style": "电影写实 / 动漫 / 2D插画 / ...",
    "color_grading": "去饱和冷色调配暖色金色点缀",
    "lighting_mood": "戏剧性侧光，高对比度",
    "film_reference": "王家卫/芬奇/某电影风格",
    "quality_tags": "4K高清，电影质感，浅景深",
    "aspect_ratio": "16:9",
    "constraints": "面部稳定不变形，画面稳定不闪烁，动作自然流畅"
  }
}
```

### 使用方式

将风格圣经存入项目创作方案，在每个提示词末尾自动附加：

```
[具体场景提示词]，
[风格圣经.color_grading]，
[风格圣经.lighting_mood]，
[风格圣经.quality_tags]，
[风格圣经.constraints]。
```

### 示例：霸道总裁剧的风格圣经后缀

**中文**：
```
电影质感，去饱和冷色调配暖色金色点缀，
戏剧性侧光高对比度，4K高清，
浅景深，面部稳定不变形，画面稳定不闪烁，
动作自然流畅不僵硬。
```

**英文**：
```
cinematic noir style, desaturated cool tones with warm gold accents,
dramatic side lighting with high contrast, 4K Ultra HD,
shallow depth of field, maintain face consistency, stable picture,
smooth natural movements.
```

## 九、Seedance 2.0 正向约束法

**Seedance 2.0 不支持 negative prompt**，所有约束必须用正向表述。

### 完整正向约束对照表

| 想避免的问题 | 错误写法（负向） | 正确写法（正向约束） |
|-------------|----------------|-------------------|
| 模糊 | 不要模糊 / no blur | 画面清晰锐利，细节丰富 / sharp focus, crisp details |
| 变形 | 不要变形 / no deformation | 面部稳定不变形，人体结构正常 / anatomically correct proportions |
| 多余肢体 | 不要多手 / no extra limbs | 人体比例自然，正常双臂双腿 / natural anatomy, exactly two arms and legs |
| 水印/文字 | 不要水印 / no watermark | 画面干净，不带文字 / clean image, no text overlay |
| AI塑料感 | 不要假 / no plastic | 皮肤纹理自然有毛孔细节 / natural skin texture with pores |
| 平光 | 不要平光 / no flat light | 有方向感的戏剧性光线 / dramatic directional lighting with clear shadows |
| 背景杂乱 | 不要乱 / no mess | 背景干净有序，构图清晰 / clean intentional background composition |
| 画面抖动 | 不要抖 / no shake | 画面稳定如三脚架 / tripod-stable, stable picture |
| 角色漂移 | 不要变脸 / no face change | 面部稳定不变形，同一角色服装一致发型不变 / maintain face and clothing consistency |
| 动作僵硬 | 不要僵硬 / no stiff | 动作自然流畅不僵硬 / smooth natural movements |

## 十、常见提示词失败原因与修复

| 失败原因 | 表现 | 修复方法 |
|---------|------|---------|
| 描述太模糊 | 生成结果随机 | 用具体可观察细节替换主观形容词 |
| 缺少光影指令 | 画面平淡 | 补充光线类型、方向、色温 |
| 矛盾指令 | 画面混乱 | 每个提示词选择一个明确方向 |
| 风格锚点缺失 | 多张图风格不统一 | 使用项目风格圣经后缀 |
| 角色特征遗漏 | 角色外貌不一致 | 每次完整列出 5+ 核心特征 + @参考图 |
| 构图不明确 | 主体位置不受控 | 明确指定景别和镜头焦距 |
| 缺少约束词 | 变形、闪烁 | 末尾焊死稳定性约束词 |
| 提示词过长 | 模型忽略后半段 | 核心信息放在前 30 词，控制总长 |
| 缺少运动描述 | 视频静止感强 | 至少一个主体运动 + 一个运镜描述 |
| 动作太剧烈 | 穿模、变形 | 改为缓慢、自然、流畅的动作 |
