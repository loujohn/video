# 分镜设计与提示词指南（Seedance 2.0 适配）

## 一、分镜粒度原则

### 核心理念：一个分镜 = 一段可生成的视频

Seedance 2.0 支持单次生成 4-15 秒视频，且内置多镜头叙事能力。**分镜不应按传统单镜头拆分，而应以「一段完整叙事节拍」为单位**。

| 分镜粒度 | 时长 | 适用场景 | 说明 |
|----------|------|----------|------|
| 单动作分镜 | 4-7s | 反应镜头、空镜头、情绪特写 | 一个明确的动作或瞬间 |
| 叙事段落分镜 | 10-15s | 对话、对峙、追逐、揭晓 | 包含 2-3 个镜头切换的完整叙事段 |

### 分镜拆分决策

```
剧本一场戏 → 拆为 2-4 个分镜（非 8-12 个单镜头）
                ↓
每个分镜 = 一次 Seedance 2.0 生成
                ↓
分镜内可包含多个镜头切换（用 Cut to / 切换到 标记）
```

**错误做法**：把一场 30 秒的对峙戏拆成 10 个 3 秒镜头
**正确做法**：拆成 2-3 个叙事段落，每段 10-15 秒，段内包含镜头切换

## 二、镜头类型

### 特写（近景 / Close-up）

**用途**：强调情绪、关键道具、微表情。

**适用场景**：眼泪、冷笑、戒指、合同签字、手机屏幕。

**中文提示词**：`面部特写`、`手部近景`、`物品微距特写`

**英文提示词**：`extreme close-up on face`, `tight shot of hands`, `macro detail of [object]`

---

### 中景（Medium Shot）

**用途**：人物上半身，兼顾表情与肢体语言。

**适用场景**：对话、对峙、日常互动。

**中文提示词**：`中景，腰部以上`、`双人中景对话`

**英文提示词**：`medium shot from waist up`, `two-shot dialogue framing`

---

### 全景（Wide Shot）

**用途**：环境与人物关系，建立空间感。

**适用场景**：场景开场、多人对峙、身份揭晓时的全场反应。

**中文提示词**：`大全景建立镜头`、`超广角全景展示完整环境`

**英文提示词**：`wide establishing shot`, `extreme wide shot showing full environment`

---

### 主观镜头（POV）

**用途**：以角色视角呈现，增强代入感。

**中文提示词**：`第一人称视角`、`过肩镜头`

**英文提示词**：`first-person POV`, `over-the-shoulder shot`

---

### 建立镜头（Establishing Shot）

**用途**：交代场景、时间、氛围。每场戏开场或转场时使用。

**中文提示词**：`全景建立镜头，展示[地点]`、`航拍城市[时段]`

**英文提示词**：`wide establishing shot of [location]`, `aerial view of city at [time]`

---

## 三、镜头运动与 Seedance 2.0 运镜词汇

Seedance 2.0 对专业摄影术语识别能力极强，以下运镜词可直接在提示词中使用。

### 运镜速查表

| 运动方式 | 情绪暗示 | 中文写法 | 英文写法 |
|----------|----------|----------|----------|
| 推镜 | 聚焦、压迫、逼近真相 | 缓慢推镜，从中景推至特写 | `slow dolly-in from medium to close-up` |
| 拉镜 | 疏离、格局、落幕 | 镜头缓缓拉远，展现完整场景 | `slow dolly-out to reveal full scene` |
| 横摇 | 巡视、发现、连接 | 镜头平稳从左向右横移 | `smooth pan from left to right` |
| 横移跟拍 | 跟随、参与感 | 平稳横移跟拍 | `steady track alongside subject` |
| 环绕 | 仪式感、全貌展示 | 环绕主体半圈 | `orbit 180 degrees around subject` |
| 升降 | 格局、命运、俯瞰 | 镜头从地面缓缓升起 | `crane up from ground level` |
| 手持 | 紧张、真实、不安 | 手持镜头，轻微晃动 | `handheld camera, natural slight shake` |
| 固定 | 对话、情感沉淀 | 固定镜头，无运动 | `static camera, locked off, no movement` |
| 稳定器 | 精致、高级感 | 稳定器平滑运镜 | `gimbal smooth movement` |

### 关键运镜原则

1. **每个分镜只选一种主要运镜**：避免矛盾指令
2. **运镜速度要与情绪匹配**：紧张用快速，抒情用缓慢
3. **优先使用舒缓运镜**：`缓慢`、`平稳`、`丝滑流畅` 出片率最高
4. **复合运镜按时间拆分**：`开头缓慢推镜，最后2秒轻微横摇`

## 四、分镜节奏控制

### 时间节拍分配模板

根据场景情绪选择节拍模板：

| 节奏类型 | 时间分配（15s） | 适用场景 |
|----------|----------------|----------|
| 慢节奏 | 5s 铺垫 + 5s 主体 + 5s 收尾 | 情感高潮、离别、回忆 |
| 中节奏 | 3s 建立 + 7s 核心 + 5s 定格 | 日常对话、身份揭晓 |
| 快节奏 | 2s 起势 + 10s 密集 + 3s 爆发 | 对峙、追逐、打脸 |

### 30 秒短视频节奏（拆为 2 个分镜）

| 分镜 | 时间段 | 功能 | 建议内容 |
|------|--------|------|---------|
| 分镜 1 | 0-15s | 冲突建立 → 矛盾升级 | 特写/主观开篇 → 建立镜头 → 正反打 |
| 分镜 2 | 15-30s | 高潮爆发 → 反转定格 | 快切 + 手持镜头 → 特写定格 |

### 60 秒短视频节奏（拆为 4 个分镜）

| 分镜 | 时间段 | 功能 | 建议内容 |
|------|--------|------|---------|
| 分镜 1 | 0-15s | 钩子 + 建立情境 | 冲突特写 → 场景全景 → 人物关系中景 |
| 分镜 2 | 15-30s | 矛盾升级 | 正反打对话 → 特写交替 → 节奏加快 |
| 分镜 3 | 30-45s | 高潮爆发 | 快切 + 手持 → 情绪爆点特写 |
| 分镜 4 | 45-60s | 反转/升华 + 悬念 | 全景或特写定格 → 悬念留白 |

## 五、剧本场景到分镜的映射

### 场景类型与分镜策略

| 剧本元素 | 分镜策略 | 分镜内镜头安排 |
|----------|----------|---------------|
| 冲突对话 | 1 个 10-15s 叙事段 | 中景 → Cut to 特写A → Cut to 特写B |
| 身份揭晓 | 1-2 个段落 | 段1：全景建立 → 众人反应特写；段2：主角中景/特写 |
| 打脸瞬间 | 1 个 10s 段落 | 反派特写（震惊）→ Cut to 主角中景（从容）→ 全景 |
| 情感高潮 | 1 个 10-15s 长镜头 | 单一缓慢推镜，不切镜 |
| 悬念结尾 | 1 个 5-7s 段落 | 特写定格或主观镜头 |
| 追逐场景 | 2 个段落 | 段1：手持跟拍追逐；段2：结果特写 |

## 七、镜头切换语法

### Seedance 2.0 支持的转换词

在分镜内部标记镜头边界，使用以下明确的转换提示词：

| 中文写法 | 英文写法 | 效果 |
|----------|----------|------|
| 切换到 | Cut to | 硬切，最常用 |
| 过渡到 | Transition to | 柔和过渡 |
| 转到 | Shift to | 视角/场景切换 |
| 淡入淡出 | Fade to black, then fade in | 时间跳跃 |
| 溶解到 | Dissolve to | 回忆、梦境 |

**关键**：如果不使用转换词，Seedance 2.0 会尝试在不同构图之间创建平滑运镜，可能导致不自然的中间帧。

### 多画面描述格式

推荐使用「第 N 个画面」格式按时间顺序描述镜头变化：

```
第一个画面：[景别]描述...
切换到 —
第二个画面：[景别]描述...
切换到 —
第三个画面：[景别]描述...
```


## 九、提示词质量约束

### Seedance 2.0 必加约束词

每个分镜提示词末尾必须附加以下约束：

**中文约束词（推荐）：**
```
面部稳定不变形，人体结构正常，比例自然，
动作自然流畅不僵硬，画面稳定不闪烁，
4K高清，细节丰富，色彩自然。
```

**英文约束词：**
```
Maintain face and clothing consistency, no distortion,
natural human anatomy, smooth natural movements,
stable picture, no flickering, 4K Ultra HD, rich details.
```

### 权重分布原则

Seedance 2.0 对提示词有位置权重偏好：

- **开头**（权重最高）：主体描述——谁/什么，核心外貌特征
- **中前段**：动作描述——连续、具体的物理运动
- **中段**：运镜和镜头语言
- **中后段**：光影、环境、风格
- **末尾**：约束词 + 画质标签

### 提示词长度控制

| 分镜类型 | 推荐词数 | 说明 |
|----------|---------|------|
| 单动作（4-7s） | 30-80 词 | 简洁明确 |
| 叙事段落（10-15s） | 80-200 词 | 每个画面 30-60 词 |

**过短**：信息不足，生成随机
**过长**：模型忽略后半段细节

### 动作描述原则（Seedance 2.0 专项）

**核心原则**：动作写慢、写连续、写具体。

| 好的写法 | 差的写法 |
|----------|---------|
| 缓缓转身，目光从窗外移向门口 | 转身 |
| 轻轻抬起右手，指尖拂过桌面 | 碰桌子 |
| 微风拂过，头发轻轻飘动，裙摆自然摆动 | 风吹 |
| 缓慢站起，双手撑住桌沿 | 站起来 |

**优先使用的动作修饰词**：缓慢、轻柔、连贯、自然、流畅、平稳、微微

**避免的描述**：夸张、高速、剧烈、复杂多人互动、大幅度扭曲

**物理感知描述**：加入材质和物理细节可提升真实感
```
丝绸裙摆随风轻轻飘动，较重的裙边自然下垂
雨滴落在肩上，湿润的织物贴紧皮肤
```

### 多镜头一致性检查

连续分镜之间需保持以下一致：
- 角色服装、发型不变（除非剧情要求换装）
- 光线方向一致（同一场景内日光/灯光方向不变）
- 色调一致（使用项目风格圣经后缀保证）
- 环境连贯（同一场景不同角度的背景元素应匹配）

## 十、分镜衔接与转场

### 转场类型速查

| 类型 | 代码 | 效果 | 适用场景 | 视频提示词处理 |
|------|------|------|----------|----------------|
| 硬切 | `cut` | 画面直接跳转 | 同场景连续动作、对话正反打 | 下一段开头写「承接上镜...」 |
| 溶解 | `dissolve` | A画面渐隐B画面渐显 | 同场景时间推移（黄昏→夜晚） | 当前段结尾写「画面渐渐模糊」 |
| 渐黑 | `fade_black` | 画面渐暗至全黑再亮起 | 不同场景跳转、集与集之间 | 当前段结尾写「镜头缓缓暗下」 |
| 渐白 | `fade_white` | 画面渐亮至全白 | 情感高潮、觉醒、回忆 | 当前段结尾写「强光笼罩画面」 |
| 推移 | `wipe` | 新画面从一侧推入 | 平行叙事切换 | 无需特殊处理 |
| 匹配剪辑 | `match_cut` | 构图/动作一致的两个画面 | 对比/关联（A转身→B转身） | 两段使用相似构图描述 |

### 转场判断逻辑

```
前后场景相同？
  ├─ 是 → 时间连续？
  │    ├─ 是 → cut（硬切）
  │    └─ 否 → dissolve（溶解）
  └─ 否 → 是否为情感爆发点？
       ├─ 是 → fade_white（渐白）
       └─ 否 → 是否为平行叙事？
            ├─ 是 → wipe（推移）
            └─ 否 → fade_black（渐黑）
每集最后一个分镜 → fade_black
```

### 衔接在视频提示词中的表达

**硬切（cut）** — 下一分镜的 prompt 开头体现连续性：
```
承接上一个画面中角色的回头动作，镜头切到角色正面特写，目光中带着惊讶...
```

**溶解（dissolve）** — 当前分镜的 prompt 结尾体现过渡：
```
...镜头缓缓拉远，画面逐渐柔和模糊，天边从橘黄渐变为深蓝...
```

**渐黑（fade_black）** — 当前分镜的 prompt 结尾：
```
...角色的背影渐渐被黑暗吞没，画面缓缓暗淡至漆黑。
```

**渐白（fade_white）** — 当前分镜 prompt 结尾：
```
...剑身爆发出刺眼白光，光芒吞噬整个画面。
```

### 节奏与转场的配合

| 场景情绪 | 推荐转场组合 | 节奏特点 |
|----------|-------------|----------|
| 紧张打斗 | cut + cut + cut | 快节奏，密集切换 |
| 日常对话 | cut + dissolve | 中等节奏，舒缓过渡 |
| 悬念铺垫 | dissolve → fade_black | 慢节奏，留白渲染 |
| 高潮爆发 | cut + fade_white | 先快后炸，情感释放 |
| 多线并进 | wipe + wipe + cut | 空间跳跃，时间同步 |

## 十一、分镜自检清单

### 叙事层面
- [ ] 每场戏是否有建立镜头？
- [ ] 关键情绪是否有特写支撑？
- [ ] 分镜粒度是否合理（每个分镜 5-15s，非 1-3s）？
- [ ] 节奏是否符合剧本情绪曲线？
- [ ] 镜头切换是否使用了明确的转换词？

### 提示词层面
- [ ] 所有角色是否使用了 @角色 标签引用参考图？
- [ ] 运镜指令是否具体（非「移动摄影机」这种模糊表述）？
- [ ] 主体描述是否在提示词开头？
- [ ] 每个分镜是否只有一个主要运镜动作？
- [ ] 动作描述是否使用了缓慢/自然/流畅等修饰词？
- [ ] 是否附加了稳定性约束词（面部不变形、画面稳定等）？
